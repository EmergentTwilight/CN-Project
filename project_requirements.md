# 复现Breaking

实验目的：
本实验旨在复现《Breaking the Sorting Barrier for Directed Single-Source Shortest Paths》。当今的大规模网络系统在路由选择中高度依赖最短路径算法，以确保低延迟与高吞吐。然而，传统的单源最短路径计算仍受限于排序开销，导致在实时网络优化中效率不足。随着数据中心和SDN网络规模的扩大，打破这一“排序瓶颈”的需求变得愈发迫切。现有算法如 Dijkstra 在比较-加法模型下已接近最优，但其排序过程限制了进一步提升性能。为克服这一局限，Duan 等人提出了新的确定性算法，通过引入分治与前沿缩减机制，首次在有向图中突破了传统排序界限，为未来高性能网络路由计算和智能调度提供了新的理论基础。具体目标如下：
1. 了解相关背景知识，掌握论文原理。
2. 基于该paper提供的方法思路，复现论文。
3. 分析论文的优点和局限性。
 
实验要求：
1. 认真阅读论文，掌握方法原理，并基于NS3等网络仿真平台在路由选择上进行复现，将论文的算法与Dijkstra算法进行路由收敛的正确性和所需时间的对比。
2. 分析论文的优点和局限性，尝试去探索一些改进方法（拓展）。
